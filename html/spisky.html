Тег
− це спеціальний символ розмітки, який застосовується для вставки різних елементів на веб-сторінку таких як: малюнки, таблиці, посилання та ін., і для зміни їх виду.
HTML-документ
− це звичайний текстовий файл, який може містити в собі текст, теги і стилі. Зображення та інші об'єкти зберігаються окремо. Вміст такого файлу зазвичай називається HTML-кодом.
Сайт
− це набір окремих веб-сторінок, які пов'язані між собою посиланнями і єдиним оформленням.
  1. Для невпрорядкованого списку викуристовуємо контейнер <ul>, для кожного пункту, використовуюмо тег <li>.
    Для впорядкованного списку використовуємо відповідно контейнер <ol>, та для пунктів також використовуюємо тег <li>.
  2. Атрибути можна додавати для "маркованого списку"
    -для маркеру у вигляді кола вокористовує атрибут disc (приклад: <ul type = "disc">)
    -для маркеру у вигляді прозорого кола використовуємо атирибут circle (приклад: <ul type = "circle">)
    -для квадратного маркеру використовуємо атрибут square (приклад: <ul type = "square">)
  3. Для нумерованого списку, щоб почати нумерацію, не з першої цифри, використовуюмо атрибут start (приклад: <ol start = "3">).
  4. Атрибут type тега <ul> відповідає за тип маркеру списку.
